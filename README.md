This service is built for file data downloading purpose. It's built using the FastAPI python framework.

## Installation

### Install requirements

`pip install -r requirements.txt`

Run the service with uvicorn
`python app.py`

### Docker

*docker-compose*

`docker-compose build`
`docker-compose up`
